# Hello World in 400+ Programming Languages

This project contains "Hello, World!" programs written in over 400 different programming languages and file formats, each saved with their appropriate file extensions.

## About This Collection

This is one of the most comprehensive collections of "Hello, World!" programs available, featuring:
- **400+ programming languages** and file formats
- **Historical languages** (ALGOL, COBOL, Fortran)
- **Modern languages** (Rust, Go, Swift, Kotlin)
- **Esoteric languages** (Brainfuck, Chef, Malbolge)
- **Domain-specific languages** (SQL, PostScript, VHDL)
- **Assembly languages** (x86, ARM, etc.)
- **Scripting languages** (Shell, PowerShell, Batch)

## Running the Programs

Each program can be executed using its respective compiler or interpreter. Here are some examples:

### Popular Languages
```bash
# C
gcc hello.c -o hello && ./hello

# C++
g++ hello.cpp -o hello && ./hello

# Python
python hello.py

# JavaScript
node hello.js

# Java
javac Hello.java && java Hello

# Go
go run hello.go

# Rust
rustc hello.rs && ./hello

# Swift
swift hello.swift

# Kotlin
kotlinc hello.kt -include-runtime -d hello.jar && java -jar hello.jar
```

### Web Languages
```bash
# HTML - Open in browser
open hello.html

# TypeScript
tsc hello.ts && node hello.js

# PHP
php hello.php
```

### Scripting Languages
```bash
# Shell/Bash
bash hello.sh

# PowerShell
powershell -File hello.ps1

# Batch (Windows)
hello.bat

# Perl
perl hello.pl

# Ruby
ruby hello.rb
```
